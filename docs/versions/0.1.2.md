# Version 0.1.2

## Backend & Graph Visualization Updates
- **Initial Probing & Seeding**: Queried the internet for context regarding Anthropic's "Project Glasswing". Utilizing the pre-existing CLI mechanism, deployed three cluster-node mappings into the Neon database focusing on Anthropic, Microsoft, and Google under a defensive AI umbrella payload.
- **Continuous Graph Flow Layout**: Overhauled `GraphCanvas.tsx` to abandon standard static assignments in favor of `useWorkerLayoutForceAtlas2` from `@react-sigma/layout-forceatlas2`. Rendering now repels nodes dynamically via a persistent background worker, allowing user drags to recalculate relative positions natively for maximum distinction.
- **Dynamic API Bridge**: Completed previous `TASK.md` backlog by migrating data delivery off front-end scaffolding mock functions to a formal server-side `/api/graph` `GET` route mapping the live Prisma Neon models cleanly to the Sigma pipeline.
