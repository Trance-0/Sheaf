# Version 0.1.1

## Bug Fixes & UI Enhancements
- **Theme Mixing Resolution**: Updated `.sigma-container` in `globals.css` with a transparent background block. This averts the raw white canvas layout from crashing with the standard dark mode application structure, fulfilling the seamless glassmorphic requirement.
- **Node Dragability**: Implemented `EventsHandler` within `GraphCanvas.tsx` integrating proper `mousedown`, `mousemove`, and `mouseup` hooks leveraging `@react-sigma/core` API `sigma.setSetting("enableCameraPanning", false)`. Users can now definitively click, drag, and reorganize node structures to their visual preferences.
